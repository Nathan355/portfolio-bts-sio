# Documentation - Dossier des Projets d'Alternance

Ce dossier contient la documentation technique pour les projets réalisés en alternance chez Openvalue.

## Fichiers

### installation-repeteur-wifi.pdf
Document PDF détaillant les étapes d'installation et de configuration du répéteur WiFi.

**Instructions pour ajouter le fichier PDF:**
1. Placez votre fichier PDF `installation-repeteur-wifi.pdf` dans ce dossier (`docs/`)
2. Le fichier apparaîtra automatiquement dans la page alternance.html
3. Les utilisateurs pourront le consulter directement dans le navigateur ou le télécharger

## Format de Référence

Pour ajouter d'autres documents:
- Placez-les dans le dossier `docs/`
- Mettez à jour le fichier `alternance.html` avec les liens appropriés
- Les PDF, images et autres ressources seront accessibles depuis `docs/votre-fichier`
